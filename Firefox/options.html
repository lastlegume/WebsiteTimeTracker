General Settings



Choose which buttons appear at the top of the extention popup window. Data will still be collected for unselected options here, but the option to visualize them will not be visible in the popup. To disable collection, see the section below.

Data collected

Warning: when disabled, no data will be collected, meaning that if reenabled, anything that would have been collected while disabled will be missing. To delete collected data, see below.



Import/Export/Clear Data

It is highly recommended that you export any data before importing new data to avoid losing any information. Imported data will be added to existing local data. Thus, repeated imports will add the same time data multiple times.

Warning: these cannot be undone. Please be absolutely sure before proceeding.
